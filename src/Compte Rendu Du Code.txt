  La classe de l'interface principale : VoteStat -> fenêtre de l'application
  interfaces principale et la plus pertinante dans le code : VotePane
  
  Bibliothèques spéciales utilisé dans le code : 
        -> JFreeChart , pour la création des graphes 
        -> JFileChooser , pour la sauvegarde des fichiers sous un certain repertoire
        -> itext , pour la sauvegarde sous le format pdf 
        -> mysqli , pour la connexion avec la base de donnée 
 
 L'animation des composant et des transitions est complètement codé intuitivement sans l'aide de bibliothèques d'animation ( la partie la plus difficile du code) 
 J'ai crée les algorithmes de manière à ce qu'ils soient clairement compris et efficaces en même temps 
 
 J'ai diviser les package d'une manière à faciliter la compréhension de la dispostion et de l'architecture du logiciel
